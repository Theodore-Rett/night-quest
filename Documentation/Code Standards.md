# Code Standards
The code conventions and standards used in this project when developing. 

The **File Structure** section outlines where things in the project are sorted, and how to find things. Maintaining a consistent system.

**Coding Standards** outlines the expected conventions used when coding. Including but not limited to variable naming conventions, function naming conventions, and general code formatting. This is to ensure that the code is consistent and easy to read for all developers working on the project.
